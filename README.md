# AMD-SLINGSHOT

Real-Time Deepfake Detection System that can instantly identify whether a video, image, or live webcam feed is real or AI-generated using deep learning and AMD accelerated computing.  The system analyzes facial features frame-by-frame and detects manipulation artifacts using AI models running on AMD GPUs accelerated by ROCm. It then displays the result as REAL or FAKE along with a confidence score through a user-friendly dashboard.  This solution helps prevent the spread of misinformation, identity fraud, and fake digital content by enabling fast, accurate, and real-time verification. It also demonstrates the power and efficiency of AMD hardware for real-world AI applications.

Our solution enables real-time detection of deepfake videos and images, helping prevent the spread of misinformation, identity fraud, and manipulated digital content. By leveraging AMD GPU acceleration through ROCm and processors from AMD, it provides fast, accurate, and scalable verification. This can be used by social media platforms, news agencies, and cybersecurity systems to improve digital trust, enhance security, and protect individuals and organizations from AI-generated fake media.
